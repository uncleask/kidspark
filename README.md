# KidSpark

> KidSpark —— 童心闪耀，发现孩子的心灵智慧，点燃孩子的创作火花。AI 驱动的童创素材库，让涂鸦、故事、录音变成动画与绘本。

这是一个为家长设计的本地素材管理工具，用于管理孩子的涂鸦、照片、故事录音和视频成品，并通过 AI 能力将简笔画自动上色、生成动态视频。

## 功能特点

### 素材管理
- **素材导入**：支持拖拽上传单个文件、多个文件或整个文件夹
- **多格式支持**：图片 (jpg/png/heic/webp)、视频 (mp4/mov/avi)、音频 (mp3/m4a/wav)
- **重复检测**：按文件名自动检测重复导入
- **批量删除**：首页素材列表支持多选批量删除，同时清理物理文件
- **自动分类**：按年/月/日组织导入的素材
- **缩略图生成**：自动为图片和视频生成缩略图

### AI 创作
- **AI 自动上色**：调用 AI-Agent 的图片生成功能，将简笔画转为精美的彩色图片
- **AI 视频生成**：从彩色图生成动态视频，让作品活起来
- **版本链管理**：支持主线串联和衍生分支，形成完整的创作版本树
- **生成记录管理**：AI 生成列表支持软删除和主线状态标记

### 素材编辑
- **图片旋转**：支持顺时针/逆时针旋转，保存后自动更新缩略图
- **描述编辑**：支持为素材添加和编辑描述文字
- **提示词管理**：记录和管理 AI 生成时使用的提示词

### 标签与筛选
- **标签系统**：为素材添加多个标签，支持多标签同时筛选
- **标签管理**：支持删除标签（带确认框），自动清理关联关系
- **类型筛选**：按图片/视频/音频类型快速筛选
- **搜索功能**：按文件名或标签搜索素材，支持组合筛选

### 导出功能
- **复制文件路径**：一键复制选中文件路径到剪贴板
- **导出到文件夹**：将素材复制到用户选择的文件夹
- **生成 JSON 元数据**：导出包含完整信息的 JSON 文件

### 样例展示
- 内置简笔画 → AI 彩色图 → CSS 动画视频的完整样例演示

## 技术栈

- **Electron**：桌面应用框架
- **React 18**：前端界面
- **TypeScript**：类型安全
- **Ant Design 5**：UI 组件库
- **SQLite (better-sqlite3)**：本地数据库
- **Sharp**：图像处理和缩略图生成
- **FFmpeg**：视频缩略图生成
- **Vite**：构建工具
- **electron-builder**：应用打包

## 安装依赖

```bash
npm install
```

## 开发模式

同时启动 Vite 开发服务器和 Electron 应用：

```bash
npm run electron:dev
```

## 构建

构建前端和 Electron 主进程代码：

```bash
npm run build
```

## 运行构建后的应用

```bash
npm run electron
```

## 打包发布

构建安装包和便携版：

```bash
npm run dist
```

打包产物位于 `release/` 目录下（以版本 1.0.0 为例）：
- `KidSpark Setup 1.0.0.exe` — Windows 安装包
- `kidspark-1.0.0-portable.exe` — 便携版（无需安装，双击即可运行）

## 使用说明

### 基础操作
1. 点击顶部「导入素材」按钮，选择需要导入的文件或文件夹
2. 素材会自动拷贝到 `~/Documents/KidSpark/originals/` 目录
3. 在素材卡片上可以添加和移除标签
4. 在左侧标签列表中点击标签可筛选素材（支持多标签组合筛选）
5. 点击标签上的「×」可删除标签（带确认框）
6. 勾选素材卡片左上角复选框，可进行批量删除

### 详情页操作
1. 点击素材卡片打开详情页
2. **左侧**：展示该素材的所有版本（原始图 + AI 生成图）
3. **中间**：大图预览 + 旋转/保存/AI生成等操作按钮
4. **右侧**：标签管理、描述编辑、提示词查看
5. 点击「AI 自动上色」生成彩色图片
6. 点击「生成视频」从彩色图生成动态视频
7. 使用「设为主线」串联创作版本链

### AI 导出功能
KidSpark 提供三种导出方式，方便将素材用于 AI 创作：

1. **复制文件路径到剪贴板**
   - 一键复制所有选中文件的路径
   - 可直接粘贴到 ComfyUI/Stable Diffusion 等 AI 工具中

2. **导出到指定文件夹**
   - 将素材文件复制到用户选择的文件夹
   - 保持原文件名

3. **生成 JSON 元数据**
   - 导出包含文件名、路径、标签、创建时间等完整信息的 JSON 文件
   - 用于批量处理和自动化工作流

### 搜索功能
- 使用顶部搜索框按文件名或标签搜索素材
- 搜索结果实时更新
- 支持与标签筛选同时使用

## AI 创作流程

```
简笔画导入 → AI 自动上色 → 生成动态视频 → 版本管理
     ↑                                    ↓
     └──────── 持续迭代优化 ←─────────────┘
```

1. **导入原始简笔画**：小朋友自己绘制的简笔画，或者扫描上传
2. **AI 自动上色**：调用 AI-Agent 的图片生成功能，生成精美的彩色图片
3. **生成 AI 视频**：从彩色图生成动态视频，让作品活起来
4. **保存和管理**：所有版本都保存，可以随时查看和管理
